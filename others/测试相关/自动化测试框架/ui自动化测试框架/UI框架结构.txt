ui_test_framework/
├── base/
│   ├── base_page.py  # 基础页面类
├── pages/
│   ├── login_page.py  # 登录页面类
│   ├── change_password_page.py  # 修改密码页面类
│   ├── comment_page.py  # 评论页面类
│   ├── chat_page.py  # 聊天页面类
├── tests/
│   ├── test_login.py  # 登录测试用例
│   ├── test_change_password.py  # 修改密码测试用例
│   ├── test_comment.py  # 评论测试用例
│   ├── test_chat.py  # 聊天测试用例
├── utils/
│   ├── browser.py  # 浏览器操作工具类
│   ├── logger.py  # 日志工具类
│   ├── retry_decorator.py  # 重试装饰器
│   ├── dingtalk_notifier.py  # 钉钉通知工具类
├── conftest.py  # pytest 配置文件
├── pytest.ini  # pytest 配置文件

BasePage 类：封装了一些常用的页面操作方法，如查找元素、点击元素、输入文本等，提高代码的复用性。
LoginPage 类：继承自 BasePage 类，定义了登录页面的元素定位和操作方法，将页面元素和操作逻辑分离。
test_login 测试用例：使用 pytest 编写测试用例，调用 LoginPage 类的方法完成登录操作。
browser.py：封装了获取浏览器驱动的方法，方便管理浏览器的启动和关闭。
conftest.py：使用 pytest 的 fixture 功能，在每个测试用例执行前后启动和关闭浏览器。
