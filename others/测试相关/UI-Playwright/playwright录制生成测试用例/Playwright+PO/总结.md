

# 目标：
如何将 Playwright 录制的脚本一键转换为 PO 模式，并兼容 pytest 框架？

本文将带你构建一个工具，利用 本地部署的 DeepSeek 模型，结合 Python 爬虫技术，快速完成以下任务：

自动提取页面控件（如按钮、输入框等），生成对应的 Page Object 类。
将 Playwright 录制的代码转换为基于 PO 模式的 pytest 测试代码。
工具生成的代码可由测试工程师进一步修改和扩展，支持复杂测试场景。

# 步骤：
录制 Playwright 脚本
使用 Playwright 录制的全屏脚本作为输入，生成待转换的测试代码。

爬取页面控件信息
利用 Python 爬虫解析 HTML，提取页面中的控件信息（如按钮、输入框等），并生成 Page Object 类。

转换为 PO 模式代码
使用 DeepSeek 模型辅助生成 pytest 测试代码，将录制脚本逐步转换为调用 Page Object 的测试用例。

输出可运行代码
输出包含 Page Object 类和 pytest 测试用例的完整项目结构，支持直接运行。
