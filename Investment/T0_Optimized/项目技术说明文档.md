# T0交易系统优化版技术说明文档

## 项目概述
T0_Optimized是基于T0交易系统的优化版本，主要改进包括：
- 模块化重构
- 指标优化
- 可视化界面增强

## 核心功能
1. 实时数据监控
2. 多策略回测
3. 可视化分析
4. 自动化交易

## 核心指标说明

### 1. 综合T0策略
- 结合量价关系和技术指标
- 动态调整交易参数
- 适用于日内波段操作

### 2. 价格均线偏离(基础)
- 计算价格与均线的偏离度
- 识别超买超卖区域
- 基础版使用简单移动平均

### 3. 价格均线偏离(优化)
- 在基础版上增加权重调整
- 动态均线周期
- 异常值处理优化

## 项目结构
```
T0_Optimized/
├── cache/            # 分时数据缓存
├── config/           # 配置文件
├── core/             # 核心交易逻辑
├── db/               # 数据库存储
├── indicators/       # 指标实现
├── output/           # 输出结果
└── visualizer_ui/    # 可视化界面
```

## 使用说明
1. 运行`run_visualizer.py`启动可视化工具
2. 选择股票和日期
3. 勾选需要分析的指标
4. 使用播放功能回放交易过程

## 优化建议
1. 调整Y轴刻度范围解决价格压缩问题
2. 确保成交量数据正确加载
3. 定期清理缓存数据
