## ✨ 有什么不同？

| 功能 | 原T0系统 | T0_Optimized |
|------|----------|--------------|
| 数据存储 | CSV文件 | **SQLite** (快10-50倍) |

## 💡 1分钟理解核心概念

### 1. DataManager = 你的数据库
```python
from core import DataManager

dm = DataManager()
df = dm.get_minute_data('000333', start_date='2025-10-24')
# 返回包含240行的DataFrame
```

**替代了什么：**
```python
# 旧方法（慢）
df = pd.read_csv('cache/fenshi_data/000333_20251024_fenshi.csv')
df['时间'] = pd.to_datetime(df['时间'])
```

### 2. IndicatorLoader = 智能插件系统
```python
from core import IndicatorLoader

loader = IndicatorLoader()
# 自动发现所有指标！

result = loader.execute_indicator('comprehensive_t0_strategy', df)
```

**替代了什么：**
```python
# 旧方法（脆弱）
try:
    from indicators.comprehensive_t0_strategy import analyze_comprehensive_t0
except ImportError:
    # 定义模拟函数...
    def analyze_comprehensive_t0(df):
        # 模拟实现
        pass
```

### 3. ConfigManager = 智能设置
```python
from core import ConfigManager

config = ConfigManager()
stocks = config.get_stocks()  # 从YAML文件读取
theme = config.get('ui.theme')  # 嵌套访问

# 轻松更新
config.set('ui.theme', 'dark')
config.save_config()
```

**替代了什么：**
```python
# 旧方法（硬编码）
STOCKS = {
    '000333': '美的集团',
    '600030': '中信证券',
    '002415': '海康威视'
}
# 需要编辑代码才能改变！
```

## ⚡ 性能示例

## ❓ 常见问题

### Q: 会影响我原来的T0系统吗？
**A:** 不会！T0_Optimized是完全独立的目录。原T0系统保持不变，可以同时使用两个系统。

### Q: 必须迁移数据吗？
**A:** 不必须。如果你只想测试新功能，可以直接使用新系统。只有当你想在新系统中使用历史数据时才需要迁移。

### Q: 迁移数据需要多长时间？
**A:** 对于242条分时数据，只需要不到1秒。批量导入1000只股票的数据约需10-30秒。

### Q: 如果出问题了怎么办？
**A:** 原始CSV文件不会被删除或修改，你可以随时回退到原系统。

### Q: 支持哪些Python版本？
**A:** Python 3.7及以上版本。推荐使用Python 3.8+。
